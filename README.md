# Eshwar Sea Foods Website

This is a static website for `www.eshwarseafoods.in` built from a single `index.html` file with image assets in the root folder.

## What was updated
- Page title updated to `Eshwar Sea Foods | www.eshwarseafoods.in`
- Added meta description, canonical URL, and Open Graph metadata
- Footer updated to include the new domain branding
- Created `CNAME` for GitHub Pages custom domain support

## Deployment options

### GitHub Pages
1. Create a repository and push this folder content.
2. Enable Pages from the repository settings.
3. Use the `CNAME` file to configure the custom domain to `www.eshwarseafoods.in`.
4. Set DNS for `www` to point to GitHub Pages.

### Other static hosts
- Deploy the files to any static hosting platform (Netlify, Vercel, Firebase Hosting, etc.).
- Configure the custom domain `www.eshwarseafoods.in` in the host dashboard.
- Add DNS records as required by the hosting provider.

## Notes
- Actual domain registration and DNS configuration must be completed by the site owner.
- If using GitHub Pages, point `www.eshwarseafoods.in` to the host and leave the `CNAME` file in place.
